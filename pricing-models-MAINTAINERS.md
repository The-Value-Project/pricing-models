# Maintainers — pricing-models

This file lists the current maintainers of the [`pricing-models`](https://github.com/The-Value-Project/pricing-models) repository.

Maintainers have merge rights on this repository. At Stage 1 of the [governance roadmap](https://github.com/The-Value-Project/.github/blob/main/GOVERNANCE.md), all maintainers are from ValueIQ Technologies Inc. When the project reaches Stage 2 (10 production implementations across the organization), an independent Technical Steering Committee will be formed and this file will be updated to reflect TSC membership.
